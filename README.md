# My Vim Shortcuts

`\f` - Toggle NERDTree
`\t` - Toggle terminal

`K` - Show documentation for object under cursor
`gd` - Goto defintion
`gi` - Goto implementation
`\rn` - Rename symbol
`CTRL-<space>` - Start completion in insert mode
